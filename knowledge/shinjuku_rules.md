**district_id:** `shinjuku`

## Burnable — 燃やすごみ (сжигаемый бытовой мусор)

**Что это:** Отходы, которые **не** идут в ресурсы, **не** в 金属・陶器・ガラス, **не** в крупногабарит.

| Примеры предметов / материалов | Примечания |
|-------------------------------|------------|
| Остатки еды, губка для мытья посуды (сильно загрязнённая), влажные салфетки | Влажные отходы: по возможности удалить лишнюю влагу |
| Бумага, сильно загрязнённая или жирная (невозможно сдать как макулатуру) | Чистую бумагу — в отдельные категории ресурсов ниже |
| Резина, изделия из кожи | Отдельно от пластика-ресурсов |
| Деревянные/тканевые предметы малых размеров, не подходящие под ресурс или другие потоки | Сверяться со словарём района для спорных предметов |

**Вынос:** до 8:00 дня сбора, только в отведённые места (資源・ごみ集積所). Частота для многих районов — **около 2 раз в неделю** (уточнять по адресу).

---

## Non-burnable — 金属・陶器・ガラスごみ

В Синдзюку отдельная категория **«metal, ceramics, glass»** соответствует «нересурсному» негорючему потоку (не путать с отдельным японским термином «不燃ごみ», который в других городах может отличаться).

| Примеры | Примечания |
|---------|------------|
| Металлическая посуда, мелкая металлическая фурнитура | Опасные острые края — упаковать в картон, пометить «危険」 |
| Керамика, фарфор, разбитое стекло | То же про безопасную упаковку |
| Лампы люминесцентные | По возможности использовать до конца; не смешивать с другими типами без инструкции |
| Изделия из пластика **не** попадающие под ресурсный пластик | Например **крупнее ~30 см**, **толще 5 мм**, **композитные**, с примесями других материалов, загрязнение |

**Вынос:** дни сбора — **2 раза в месяц** (конкретные будни зависят от района и недели месяца).

## Resources (Plastic) — 資源プラスチック（容器包装 + 製品プラスチック）

Сюда относится **прозрачный и цветной пластик ресурсного качества**, собираемый вместе с контейнерной/упаковочной пластикой.

### Правило с апреля 2024 (令和6年4月) — критично для RAG

Раньше многие **製品プラスチック** (изделия из пластика) шли как **燃やすごみ**. С **апреля 2024**: если выполнены условия ниже, они выносятся как **ресурсный пластик** вместе с **容器包装プラスチック** (пластик с символом маркировки контейнерной упаковки — «プラマーク», когда применимо).

**Условия для «製品プラスチック»:**

| Условие | Формулировка из официальной листовки |
|--------|--------------------------------------|
| Материал | Только **100% пластик**, **один тип материала**（単一素材） |
| Размер | Примерно **в пределах 30 см × 30 см**（おおむね30㎝角以内） |
| Толщина | **Не более примерно 5 мм**（厚さ5㎜以下） |

**Примеры предметов из официальных материалов** (перечень для запросов пользователя):

- Вёдра из пластика (プラスチックのバケツ / «чёрное ведро» — если материал однородный и размер в норме → **ресурс**, а не сжигание).
- Вешалки из пластика (プラスチックハンガー).
- Зубные щётки (歯ブラシ).
- Игрушечные блоки (おもちゃ・ブロック), линейки (定規), вёдра для бани (風呂桶), герметичные контейнеры при соблюдении размеров/однородности.

**Что не класть в ресурсный пластик:**

| Исключение | Куда обычно |
|------------|-------------|
| Встроенные мини-аккумуляторы, батарейки, электроника с батареей | Не смешивать с ресурсным пластиком; уточнять пункты приёма / дни металла и опасных компонентов |
| Зажигалки | Отдельно (опасность воспламенения); **не** ресурсный пластик |
| Сильные загрязнения или остатки содержимого | Промыть по возможности; иначе другой поток или сжигание |
| Размер **свыше ~30 см** или толще лимита | **金属・陶器・ガラスごみ** по инструкции района или другой канал |

**Способ подачи:** **容器包装プラスチック** и **製品プラスチック** можно положить **в один мешок**, выносить **в день ресурсов** (**раз в неделю** — расписание без изменений по сравнению с прежней логикой ресурсного дня).

### Другие ресурсные потоки (кратко)

| Поток | Примеры |
|-------|---------|
| Бутылки / банки | Стеклянные бутылки; жесть и алюминий по правилам разделения |
| Полиэтиленовая плёнка / пакеты | По инструкции района |
| Картон / макулатура | Газеты, коробки, указанные типы бумаги |

## Sodai Gomi — 粗大ごみ (крупногабарит, платно)

**Определение:** Мебель, матрасы, крупная техника и т.п., где **сторона длиннее ~30 см** (крупный габарит из домохозяйства).

| Шаг | Действие |
|-----|----------|
| Запись | Предварительная заявка в **粗大ごみ受付センター** |
| Контакт | Интернет 24 ч.: https://www.shinjuku-sodai.com ; тел. 03-5304-8080 (пример графика в официальных материалах) |
| Оплата | Стикер/сбор оплачивается по тарифам центра |

**Юридически:** промышленный/коммерческий крупный мусор обрабатывается отдельно (ответственность предприятия, лицензированные перевозчики).

## Шпаргалка для модели (Vision → verdict)

1. Если объект — **однородный пластик**, **до ~30 см**, **тонкий**, чистый → склонять к **Resources (Plastic)** с пояснением про правило **апреля 2024**.
2. Если пластик **крупнее**, **толще**, **композит**, **грязный** или с **батареей** → не ресурсный пластик; проверять **金属・陶器・ガラス** или **燃やすごみ**.
3. Всегда напоминать пользователю уточнять **конкретный день вывоза по адресу** и использовать официальные PDF / приложение района.
